DATA STRUCTURES

n.d.

Data structures are how computer scientists store data.

List

Indexes organize the data. Repeated data is allowed. 


      public class List<T> {
        private T con[];
        private int size;
      }

Set

There can be no repeated data. May either be ordered or unordered to the client. 


      public class Set<T> {
        private T con[];
        private int size;

        /* ... */
        
        public boolean add(T item) {
          for (T curr : con) {
            if (curr.equals(item)) {
              return false;
            }
          }
          con[size++] = item;
        }
      }
